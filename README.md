# urbanFlowGen
urbanFlowGen is an open-source Python library for generating synthetic urban CFD scenarios tailored for machine learning applications. It provides a fully automated pipeline for:

    Sampling physically realistic atmospheric inflow conditions from reanalysis data,

    Generating randomized 3D urban geometries with customizable features,

    Creating computational meshes using open-source grid generation tools,

    Setting up CFD simulations for both CPU- and GPU-based solvers (e.g., OpenFOAM, m-AIA).

The library is modular and extensible, making it easy to integrate with custom solvers or simulation workflows. It is designed to support researchers and practitioners working on data-driven urban flow modeling, offering a scalable solution for producing diverse and physically realistic training datasets.
